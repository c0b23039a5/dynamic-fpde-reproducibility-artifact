# Data and Code Availability Statement

The experiments use OpenML-CC18 benchmark suite ID 99. The artifact does not redistribute OpenML datasets; the experimental runner downloads task data from OpenML at execution time.

The code used to run the experiments, the FPDE implementation, the OpenML-CC18 experimental runner, per-task/per-seed rerun scripts, and precomputed 72-task/10-seed outputs are included in this supplemental reproducibility artifact. The artifact also includes generated LaTeX tables, audit CSV files, and scripts for regenerating the manuscript tables from the precomputed per-run outputs.

The reproducibility artifact has been deposited with DOI [10.5281/zenodo.20225275](https://doi.org/10.5281/zenodo.20225275).

The public GitHub repository is available at [https://github.com/fpde-xai/fpde-reproducibility-artifact](https://github.com/fpde-xai/fpde-reproducibility-artifact).
